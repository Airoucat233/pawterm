PawTerm 交互提案

先选项目，再选 Agent，最后进入会话。

这个原型把 Claude、Codex，以及未来的 Gemini 都当成 Agent。 用户在项目边界选择执行者，每条会话保留自己的 Provider、模型、权限、工具和历史语义。

点“当前 Agent” 打开 Agent 选择器。Provider 差异集中在这个面板里，不散落到各处设置。
点“新会话” 查看启动流程。默认使用项目 Agent，但仍然可以临时换一个。
点第一条会话 进入聊天页。工具调用统一成命令、文件变更、审批等用户能理解的卡片。
